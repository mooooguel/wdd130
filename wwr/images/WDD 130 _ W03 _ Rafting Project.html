W03 Rafting Project: About Us Page

Overview

In this assignment you will create a page for the rafting site project. This page will be an "About Us" type of page which may provide contact information, history, and other promotions. The wireframe for this page is provided and you will have the opportunity to refine your work even after this initial page build and Page Evaluation.

Instructions

File Setup

  1. In VS Code, add a file named "about.html" in the wwr directory. This file is at the same level/directory as your site-plan-rafting.html file.
    If you forgot how to add a file to a specific directory, review the steps from the last assignment.

Write the CSS

  1. Create a CSS file for your rafting site named "rafting.css" and store it in the styles folder within the wwr folder.
  2. Add an @import reference from Google Fonts to use the font(s) you have selected in your site plan.
  3. Define the CSS variables in the document :root pseudo-class selector from the color schema you have selected and documented in your website plan.
  4. Begin by styling the headings and paragraph elements.
  5. Add colors to the fonts.
  6. Remove the underlines from the social media links.
  7. Consider limiting the entire width of the page to make it easier to design the content.

Style the Hero Container using Position Absolute

  1. The div with the class of hero is positioned relative in order the h1 and the article elements to be positioned on top of the hero image using absolute positioning.
  2. The hero image fits within the allocated space.
    Use a relative width CSS declaration for the hero image and make the image a block to fill the width of the container.
    Example 
    .hero img {
  display: block;
  width: 100%;
  height: auto;
}

    We use height auto in order to maintain the aspect ratio of the image which is critical. In addition, your image should be edited to be large enough to fill this space. You cannot force images to be larger than their original size or else they will be pixelated and ugly.

  3. The h1 heading is positioned absolutely.
  4. The article is positioned absolutely.
  5. The img inside the article floats to the right of the p paragraph.
  6. Consider styling the background-color and opacity of the article to fit the design and make sure it is readable.
